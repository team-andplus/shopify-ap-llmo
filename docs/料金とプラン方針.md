# AP LLMO 料金とプラン方針

壁打ちで決めた「Free / 有料」の線引きとログの扱いを記録。

---

## プラン構成

|  | Free（無料） | Paid（有料・想定 15ドル/月） |
|--|--------------|------------------------------|
| **llms.txt** | ✅ 利用可 | ✅ |
| **docs/ai** | ✅ 利用可 | ✅ |
| **AI 生成**（llms.txt 本文・再生成など） | ❌ なし | ✅ |
| **llms.full.txt 自動生成** | ❌ なし | ✅ |
| **アクセスログの見える化**（管理画面での集計表示） | ❌ なし | ✅ |

有料は 7.5 ドル案もあったが、llms.full.txt 自動生成込みで 15 ドルを想定。

---

## ログの扱い

- **無料でもログ自体は取得している**
  - App Proxy 通過時に `log/llmo-access.log` へ 1 行追記する処理は、**全ストア共通**で実行する（プランで分岐しない）。
- **「ログなし」＝「ログの見える化なし」**
  - 無料プランでは「アクセスログ」画面を出さない（または見せない）。データは記録しておき、有料になった時点からそのストアの履歴を表示できるようにする。
- 記録データは自社運用のみ。他社には見せない・売らない前提。

---

## 実装上の注意

- ログファイルへの **書き込み**: プラン未実装時は全ストアで実施してよい。
- **アクセスログ画面**（`/app/access-log`）: 有料プラン導入時に、プラン判定で表示可否を切る。
- 将来的に Billing API 等でプラン判定を入れたら、無料ストアは「アクセスログ」へのリンクを非表示 or 遷移時に「有料プランでご利用ください」とする。
