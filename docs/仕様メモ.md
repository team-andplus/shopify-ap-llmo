# AP LLMO 仕様メモ

> 最終更新: 2026-03-12

---

## 1. アプリ概要

**AP LLMO**（ANDPLUS LLMO）は、Shopify ストアの `<head>` に AI/LLM 向け文書へのリンクを追加し、AI エージェントがストア情報を適切に解釈できるようにするアプリ。

### 思想
- 「嘘をつかせない」— 事実・証拠を優先し、誇張や捏造を避ける
- AI がストア情報を適切に解釈・生成できるよう、一次情報の所在を明確にする

## 3. 管理画面の構成

### 3.1 メイン画面（/app）

#### 左カラム（メインコンテンツ）

1. **サイト情報セクション**
   - サイト種類（corporate / ec / corporate_ec）
   - タイトル（H1 用）
   - 役割・一次情報の所在
   - セクション構成メモ
   - Notes for AI

2. **llms.txt セクション**
   - AI 生成ボタン（OpenAI API 使用）
   - 対話的改善ボタン
   - テキストエディタ
   - 保存ボタン
   - CDN URL 表示

3. **llms.full.txt セクション**
   - 自動生成ボタン（全商品・コレクション・ポリシー等を取得）
   - AI 整形オプション
   - 最終生成日時表示
   - CDN URL 表示

4. **.ai-context セクション**
   - AI 生成ボタン
   - 対話的改善ボタン
   - テキストエディタ
   - 保存ボタン
   - CDN URL 表示

5. **docs/ai セクション**
   - 最大 5 件の Markdown ファイル登録
   - ファイル名 + コンテンツ入力
   - 保存ボタン

6. **OpenAI API Key 設定**
   - API Key 入力（暗号化して保存）
   - 設定状況表示

#### 右カラム（サイドバー）

1. **プラン状態バナー**
   - トライアル中: 残り日数表示 + Pro プランへのリンク
   - トライアル終了: アップグレード案内
   - Pro プラン: ステータス表示

2. **ステータスセクション**
   - llms.txt 設定状況
   - docs/ai 件数
   - アクセスログページへのリンク

3. **参考リンク**
   - andplus.co.jp の llms.txt / docs/ai 参考リンク

4. **README 構成ガイド**
   - docs/ai/README.md の推奨構成

5. **このアプリの機能**
   - 機能一覧

6. **セットアップ手順**
   - 1. サイト情報を入力
   - 2. AI で llms.txt を生成・保存
   - 3. テーマエディタで有効化

7. **週次レポート設定**
   - メールアドレス入力
   - 有効/無効切り替え
   - テストメール送信ボタン
   - 保存ボタン

8. **開発者向け**
   - 定時処理手動実行ボタン（llms.full.txt 再生成 + ログローテーション）

### 3.2 課金ページ（/app/billing）

- トライアル状態表示（残り日数）
- Pro プラン内容
  - $20/月
  - 7日間無料トライアル
  - 含まれる機能一覧
- 「Pro プランを開始」ボタン → Shopify 承認画面へリダイレクト

### 3.3 アクセスログページ（/app/access-log）

- 総リクエスト数
- AI ボット一覧（検出対象ボットとサービス名）
- ストア別集計

- 日付別集計
- AI ボット別集計
- 直近アクセス一覧（AI ボットは緑色ハイライト）

## 4. 生成されるファイル

| ファイル | 役割 | URL |
|----------|------|-----|
| llms.txt | 思想・プロトコル | /llms.txt |
| llms.full.txt | サイト情報全部（自動生成） | /llms.full.txt |
| .ai-context | AI 解釈ガイドライン | /.ai-context |
| docs/ai/*.md | 補足文書 | /docs/ai/{filename} |
| sitemap-ai.xml | AI 向けサイトマップ | /sitemap-ai.xml |

---

## 5. テーマ App Extension（LLMO head）

ストアの `<head>` に追加されるリンク:

```html
<link rel="llms" href="https://shop.myshopify.com/llms.txt" />
<link rel="llms-full" href="https://shop.myshopify.com/llms.full.txt" />
<link rel="ai-context" href="https://shop.myshopify.com/.ai-context" />
<link rel="ai-docs" href="https://shop.myshopify.com/docs/ai/README.md" />
<link rel="sitemap" type="application/xml" href="https://shop.myshopify.com/sitemap-ai.xml" />
```

テーマエディタ（カスタマイズ → アプリ埋め込み → LLMO head）で個別にオン/オフ可能。

## 6. URL リダイレクト + App Proxy

### フロー
```
/llms.txt → URL Redirect → /apps/llmo/llms.txt → App Proxy（ログ記録）→ CDN
```

- クリーンな URL でアクセス可能
- App Proxy 経由でアクセスログを記録
- sitemap-ai.xml も同様にリダイレクト設定

---

## 7. 定時処理（Cron）

| ジョブ | 実行時刻 | 内容 |
|--------|----------|------|
| 日次 | JST 2:00 AM | llms.full.txt 全ストア再生成 + ログローテーション（5000件） |
| 週次 | 月曜 JST 9:00 AM | 週次レポートメール送信 |

---

## 8. 週次レポートメール

- **送信条件**: reportEnabled = true かつ reportEmail が設定済み
- **送信内容**:
  - 総アクセス数（過去7日間）
  - AI Bot アクセス数
  - AI Bot 別アクセス（表形式）
  - ファイル別アクセス（表形式）
- **言語**: ユーザーの locale に合わせて日本語/英語
- **SMTP**: nodemailer 使用（環境変数で設定）
